# Codex Skillpack for Gemini CLI

Эта папка содержит перенос навыков из Codex в формат Gemini CLI. Навыки доступны двумя способами:

- Вручную через команды: `/frontend-vue-developer`, `/js-css-vue-review`, `/project-architect`, `/web-backend-node`.
- Автоматически: если запрос пользователя совпадает с описанием или триггерами навыка, следуй его инструкциям.

## Правила авто‑выбора навыка
1) Если запрос явно соответствует описанию навыка или его триггерам — применяй этот навык.
2) Если подходит несколько навыков — выбери минимально достаточный набор.
3) Если запрос неясен — задай уточняющий вопрос.
4) В случае конфликта инструкций следуй приоритету: system > developer > skill.

## Навыки
- **frontend-vue-developer**: генерация минималистичного Vue 3 UI (Composition API, чистый JS/CSS), responsive, анимации, доступность.
- **js-css-vue-review**: детальное ревью JavaScript/CSS/Vue кода (ошибки, практика, производительность, безопасность).
- **project-architect**: архитектура и план веб‑проекта на Vue 3 + Node.js.
- **web-backend-node**: чистый Node.js backend (core modules), REST endpoints, безопасность и производительность.

## Примечания
- Текст каждого навыка полностью сохранён в соответствующей команде `.toml`.
- При запуске команды следуй её инструкциям без отклонений.
