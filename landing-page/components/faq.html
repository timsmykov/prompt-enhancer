Frequently Asked Questions

Which platforms does the extension work on?

+

Prompt Enhancer supports all popular AI platforms: ChatGPT, Claude, Grok (X.AI), Perplexity, Google Bard and others. We're constantly adding support for new services.

Is it safe to use the extension?

+

Yes, absolutely safe. The extension works only in your browser and doesn't transmit your data to external servers. All code is open source and available for review on GitHub.

Do I need to pay to use it?

+

No, Prompt Enhancer is completely free. No subscriptions, hidden fees, or usage limitations.

How does prompt enhancement work?

+

The extension analyzes your prompt and adds structure, context, and clarifications that help AI better understand your request and provide more comprehensive responses.

When will the final version be available?

+

We plan to release the first version in the coming weeks. Waitlist subscribers will be notified first.
